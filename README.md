<strong>WCF Proxies - From beginner to expert</strong><br/>
Learn to create custom WCF Proxies using <span style="">ClientBase&lt;T&gt;</span> and <span style="">ChannelFactory&lt;T&gt;</span> classes.
<br/>
<ol>
<li>Download the source code</li>
<li>Open the solution in Visual Studio </i>(preferably as administrator)</i></li>
<li>Run WcfProxies.WindowsHost and press the Start button. If promped to allow access through Windows Firewall click OK</li>
<li>Run the WcfProxies.Clients and start testing the proxies</li>
</ol>
